# Portfolio

Modern-styled portfolio with gentle responsive features and streamlined, readabilility focused-layout. Based in Next.js, TailwindCSS, and Node.js.

## Installation and Setup

Launched with Next.js and Tailwind CSS. [View the official Next.js + Tailwind CCS setup 
docs on the Tailwind CSS website.](https://tailwindcss.com/docs/guides/nextjs)

## Topics covered

[Node.js](https://nodejs.org/en/)  
[Next.js](https://nextjs.org/)  
[TailwindCSS](https://tailwindcss.com/)   
[Conditional Rendering](https://reactjs.org/docs/conditional-rendering.html)    
[useState](https://reactjs.org/docs/hooks-state.html)  
[CSS Animations](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Animations/Using_CSS_animations)  
[Formspree](https://formspree.io/)  
[React Hooks (useState, useReducer)](https://reactjs.org/docs/hooks-effect.html)  

## Considerations

Favicon is left out - if need to generate, visit [here](https://realfavicongenerator.net/).  
Skill section folders "familiar" and "fluent" are left in but emptied of original assets.  
Pseudo-classes, pseudo-states and responsive stylings are placed at the end of each Tailwind class.
